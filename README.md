# Medical Insurance Cost Prediction System

## Overview

The Medical Insurance Cost Prediction System is a web-based machine learning application developed using Flask and Scikit-learn. It predicts an individual's estimated medical insurance cost based on demographic and lifestyle factors, including age, gender, body mass index (BMI), number of children, smoking status, and region.

The application integrates a trained machine learning model with a user-friendly web interface, enabling users to obtain insurance cost predictions in real time.

## Installation

### Clone the Repository

```bash
git clone https://github.com/Sitikanta09/Medical-Insurance-Cost-Prediction.git
```

### Navigate to the Project Directory

```bash
cd Medical-Insurance-Cost-Prediction
```

### Install Required Packages

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

The application will be available at:

```
http://127.0.0.1:5000
```
